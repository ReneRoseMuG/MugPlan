# FT27 + E2E Fixes (MasterData Defaults, Formular-Terminlisten)

## Zweck
Stabilisierung eines fehlgeschlagenen FT27-Integrationstests sowie zweier fehlgeschlagener Browser-E2E-Tests in den Terminlisten der Tour-/Mitarbeiter-Formulare.

## Scope
- Test-Setup im Reset-Pfad an Produktions-Bootstrap angeglichen (MasterData-Defaults).
- Browser-E2E für Tour-/Mitarbeiter-Formular auf korrekten Tab-Kontext (`Termine`) gebracht.
- E2E-Assertions für Scroll-Container robust gegen viewport-abhängige Geometrie gemacht.

## Technische Entscheidungen
- `resetDatabase()` seedet jetzt neben Rollen auch FT27-Default-Kategorien über `ensureMasterDataDefaults()`.
  - Begründung: `server/index.ts` macht das im regulären Startup ebenfalls; Tests sollen denselben fachlichen Grundzustand haben.
- Browser-E2E klickt vor Tabellen-Assertions explizit auf `tab-tour-termine` bzw. `tab-employee-termine`.
  - Begründung: Die Tabelle existiert nur im `Termine`-Tab.
- Im Employee-E2E wurde die Scroll-Assertion von strikt „aktuell overflowt“ auf stabile Container-Invariante umgestellt (`overflowY` ist `auto|scroll`).
  - Begründung: tatsächliche Overflow-Höhe ist viewport-/layoutabhängig und war flaky, obwohl der Innen-Scroll-Container korrekt konfiguriert ist.

## Betroffene Dateien
- `tests/helpers/resetDatabase.ts`
- `tests/e2e-browser/appointments-list.tour-employee.browser.e2e.spec.ts`

## Hinweise zum Testen
Ausgeführt (gezielt):

1. `npx cross-env NODE_ENV=test MUGPLAN_MODE=test vitest run -c vitest.integration.config.ts tests/integration/server/masterData.ft27.integration.test.ts`
   - Ergebnis: **grün** (9/9).
2. `npx cross-env NODE_ENV=test MUGPLAN_MODE=test playwright test -c playwright.config.ts tests/e2e-browser/appointments-list.tour-employee.browser.e2e.spec.ts`
   - Ergebnis: **grün** (2/2).

## Bekannte Einschränkungen
- Die Employee-E2E-Scroll-Prüfung validiert die korrekte Scroll-Container-Konfiguration, nicht zwingend „overflowt in jedem Viewport sofort“. Das ist bewusst, um flakiness durch Layout-/Viewport-Varianz zu vermeiden.
