# Projekt-Kontext — MuGPlan

## Standard-Log-Ziel

`PROJ-1`

Der Agent schreibt Schritt-Logs und MCP-Abschlusskommentare nach Auftragsklasse 4 und 5 standardmäßig an dieses Projekt im Projekt-Manager-MCP.

## Hinweise

- Spezifische Parent-Referenzen (`TKT-*`, `UC-*`, `FEAT-*`, `MS-*`) werden zusätzlich kommentiert, wenn der Auftrag explizit von einer solchen Referenz ausgeht.
- Der Skill `mcp-code-auftrag` ist maßgeblich für die MCP-Integration.
