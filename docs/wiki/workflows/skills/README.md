# Skill-Entwicklung

Dieser Ordner ist der Entwicklungsbereich für spätere repo-eigene Skill- oder Workflow-Spezifikationen.

## Regeln

- Inhalte sind zunächst Markdown-Spezifikationen.
- Es werden hier keine ausführbaren Automationen vorausgesetzt.
- Jeder Skill beschreibt Zweck, Eingaben, Ablauf, Ergebnis, Sicherheitsregeln und Abbruchbedingungen.
- Rollen- und Berechtigungsprüfungen bleiben Pflicht, wenn ein Workflow Code gegen Feature-Inhalte prüft.
