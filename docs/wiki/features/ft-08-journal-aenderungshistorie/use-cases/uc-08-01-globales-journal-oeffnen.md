# UC 08/01: Globales Journal öffnen

## Metadaten

- Feature: [FT (08): Journal / Änderungshistorie](../ft-08-journal-aenderungshistorie.md)

## Akteur

Administrator, Disponent

## Ziel

Eine globale, fachlich lesbare Übersicht der Änderungshistorie einsehen.

## Vorbedingungen

- Der Akteur ist angemeldet.
- Der Akteur besitzt Journal-Leseberechtigung.

## Ablauf

1. Der Akteur öffnet das Journal.
2. Das System lädt die neuesten Einträge absteigend.
3. Das System zeigt je Eintrag Zeit, Akteur, betroffene Entität und eine lesbare Beschreibung.

## Alternativen

Keine.

## Ergebnis

Der Akteur sieht die globale Historie.
