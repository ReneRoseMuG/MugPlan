# UC 08/03: Kontextbezogenes Journal in einem Detailformular lesen

## Metadaten

- Feature: [FT (08): Journal / Änderungshistorie](../ft-08-journal-aenderungshistorie.md)

## Akteur

Administrator, Disponent

## Ziel

Änderungen lesen, die einem konkreten Kunden, Projekt oder Termin zugeordnet sind.

## Vorbedingungen

- Ein bestehender Datensatz ist im Detailformular geöffnet.
- Der Akteur besitzt Journal-Leseberechtigung.

## Ablauf

1. Der Akteur öffnet im Detailformular den Haupttab „Journal“.
2. Das System lädt Einträge mit direktem oder abgeleitetem Bezug zum Datensatz.
3. Das System zeigt die Einträge absteigend nach Zeit an.

## Alternativen

Keine.

## Ergebnis

Die Historie ist im fachlichen Kontext sichtbar.
