# UC 26/09: Tourenplan drucken

## Metadaten

- Feature: [FT (26): Auswertungen und Reports](../ft-26-auswertungen-und-reports.md)

## Akteur

Disponent, Administrator

## Ziel

Aus dem Tourenplan-Report eine Druckausgabe erzeugen.

## Vorbedingungen

- Der Tourenplan wurde erzeugt.

## Ablauf

1. Der Akteur klickt auf „Tourenplan drucken“.
2. Das System öffnet die Druckvorschau.
3. Die Kopfzeile enthält den Tournamen.
4. Die Fußzeile enthält die Seitennummer.
5. Der Akteur wählt optional Hoch- oder Querformat.
6. Das System öffnet den Browser-Druckdialog.

## Alternativen

- Keine Einträge vorhanden: Das System erzeugt eine leere Seite mit Hinweis.

## Ergebnis

Der Tourenplan kann im gewählten Format gedruckt oder als PDF gespeichert werden.
