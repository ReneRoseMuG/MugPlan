# UC 03/06: Auslastungsansicht eines Mitarbeiters anzeigen

## Metadaten

- Feature: [FT (03): Kalenderansichten](../ft-03-kalenderansichten.md)

## Akteur

Disponent

## Ziel

Die geplante Belegung eines Mitarbeiters über vier Kalenderwochen einschätzen.

## Vorbedingungen

- Das Mitarbeiterformular ist im Bearbeitungsmodus geöffnet.

## Ablauf

1. Der Akteur öffnet den Reiter „Auslastung“.
2. Das System lädt die Termine des Mitarbeiters für den sichtbaren Vier-Wochen-Bereich.
3. Das System zeigt die Auslastung schreibgeschützt an.
4. Der Akteur navigiert bei Bedarf mit „Früher“, „Heute“ oder „Später“.

## Alternativen

Keine.

## Ergebnis

Die Auslastung ist kalenderartig sichtbar.
