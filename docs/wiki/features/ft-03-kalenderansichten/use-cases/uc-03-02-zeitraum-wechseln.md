# UC 03/02: Zeitraum wechseln

## Metadaten

- Feature: [FT (03): Kalenderansichten](../ft-03-kalenderansichten.md)

## Akteur

Disponent

## Ziel

Einen anderen Kalenderzeitraum anzeigen.

## Vorbedingungen

- Eine Kalenderansicht ist geöffnet.

## Ablauf

1. Der Akteur navigiert vor, zurück oder springt auf einen anderen sichtbaren Zeitraum.
2. Das System lädt die Termine für den neuen Zeitraum.
3. Das System rendert die gewählte Kalenderansicht neu.

## Alternativen

Keine.

## Ergebnis

Der gewünschte Zeitraum ist sichtbar.
