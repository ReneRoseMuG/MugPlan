# UC 03/05: Monatsübersicht anzeigen

## Metadaten

- Feature: [FT (03): Kalenderansichten](../ft-03-kalenderansichten.md)

## Akteur

Disponent

## Ziel

Einen Monatsbereich verdichtet überblicken.

## Vorbedingungen

- Termine sind im Zeitraum vorhanden oder der Zeitraum kann leer angezeigt werden.

## Ablauf

1. Der Akteur öffnet die Monatsübersicht.
2. Das System lädt die Termine des Monats.
3. Das System ordnet Termine je Wochenzeile in Tour-Slots ein.
4. Das System zeigt die Monatsübersicht an.

## Alternativen

Keine.

## Ergebnis

Der Monatsbereich ist in verdichteter Kalenderform sichtbar.
