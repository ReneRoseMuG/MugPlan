# UC 03/03: Darstellungsgrad der Wochenkacheln umschalten

## Metadaten

- Feature: [FT (03): Kalenderansichten](../ft-03-kalenderansichten.md)

## Akteur

Disponent

## Ziel

Die Informationsdichte der Terminkacheln in der Wochenansicht anpassen.

## Vorbedingungen

- Der Wochenkalender ist geöffnet.

## Ablauf

1. Der Akteur wählt im Bereich „Kacheln“ den Darstellungsgrad „Kompakt“, „Standard“ oder „Detail“.
2. Das System speichert die Auswahl benutzerspezifisch.
3. Das System rendert die Wochenkacheln neu.

## Alternativen

Keine.

## Ergebnis

Die Wochenansicht zeigt dieselben Termine mit angepasster Informationsdichte.
