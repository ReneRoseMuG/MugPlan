# UC 03/01: Wochenkalender anzeigen

## Metadaten

- Feature: [FT (03): Kalenderansichten](../ft-03-kalenderansichten.md)

## Akteur

Disponent

## Ziel

Die Termine einer Woche in der zentralen Planungsansicht sehen.

## Vorbedingungen

- Termine und Touren sind im System vorhanden.

## Ablauf

1. Der Akteur öffnet die Wochenplanung.
2. Das System lädt die Termine des sichtbaren Wochenbereichs.
3. Das System ordnet die Termine in Tour-Lanes und die Gruppe „Ohne Tour“ ein.
4. Das System zeigt die Termine in der Wochenansicht an.

## Alternativen

Keine.

## Ergebnis

Der Wochenkalender ist sichtbar und dient als zentrale Planungsoberfläche.
