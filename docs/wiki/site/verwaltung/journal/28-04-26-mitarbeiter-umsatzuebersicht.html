28-04-26-mitarbeiter-umsatzuebersicht: 28.04.26 | Implementierung | Mitarbeiter: Umsatzübersicht im Formular ergänzt

Datum28.04.26

Im Mitarbeiterformular wurde eine neue rein lesende Umsatzübersicht ergänzt. Sie aggregiert qualifizierte Mitarbeitertermine serverseitig auf Wochenebene, bietet einen KW-Filter und zeigt zusätzliche Wocheninformationen per Hover-Preview.

Zusammenfassung

Im Mitarbeiterformular wurde eine neue rein lesende Umsatzübersicht ergänzt. Sie aggregiert qualifizierte Mitarbeitertermine serverseitig auf Wochenebene, bietet einen KW-Filter und zeigt zusätzliche Wocheninformationen per Hover-Preview.

Betroffene Features

  • FT (05): Mitarbeiterverwaltung

Offene Punkte

Die Übersicht bleibt bewusst rein lesend und enthält keine Export- oder Schreibaktionen.
